# Product Brief: Экспресс-калькулятор печати

## Обзор

| | |
|---|---|
| **Название** | Экспресс-калькулятор стоимости печати |
| **Тип** | Web + Telegram Mini App |
| **Приоритет** | P1 (высокий) |
| **Срок** | 29.12.2024 (Web), 01.01.2025 (TG) |

---

## Проблема

Клиент спрашивает: «Сколько будет стоить напечатать деталь примерно 10×5×3 см?»

**Сейчас:** Менеджер не может ответить без файла. Говорит «пришлите модель» → клиент уходит.

**Нужно:** За 30 секунд назвать вилку цены по габаритам.

---

## Решение

Калькулятор, который по габаритам и сложности даёт **диапазон цены** (от X до Y ₽).

Работает:
- На сайте (для офиса)
- В Telegram Mini App (на выезде, с телефона)

---

## Функциональные требования

### Ввод данных
- [ ] Габариты: Длина × Ширина × Высота (мм)
- [ ] Технология: FDM / SLA / SLS / SLM
- [ ] Сложность: Простая / Средняя / Сложная
- [ ] Количество: 1, 5, 10, 50, 100+

### Расчёт
- [ ] Оценка объёма: `V = Д × Ш × В × КоэфЗаполнения`
- [ ] Вилка цены: `[Цена_мин, Цена_макс]`
- [ ] Отображение: «от 1 500 до 3 000 ₽»

### Коэффициенты заполнения

| Сложность | Заполнение | Описание |
|-----------|------------|----------|
| Простая | 15-25% | Кубик, цилиндр, простые формы |
| Средняя | 25-40% | Корпус, крепёж, типовые детали |
| Сложная | 40-60% | Органика, тонкие стенки, полости |

### Дополнительно
- [ ] Сохранение расчёта в БД
- [ ] Быстрое создание клиента (имя + телефон)
- [ ] Переход к подробному расчёту

---

## Формула расчёта

```javascript
// Оценка объёма (см³)
const габаритныйОбъём = (Д * Ш * В) / 1000; // мм³ → см³

const заполнение = {
  'простая': [0.15, 0.25],
  'средняя': [0.25, 0.40],
  'сложная': [0.40, 0.60]
};

const [заполнМин, заполнМакс] = заполнение[сложность];

const объёмМин = габаритныйОбъём * заполнМин;
const объёмМакс = габаритныйОбъём * заполнМакс;

// Вилка цены
const ценаМин = объёмМин * тариф * количество;
const ценаМакс = объёмМакс * тариф * количество;
```

## Пользовательский путь

```
1. Открывает калькулятор (Web или TG)
2. Вводит габариты (100 × 50 × 30 мм)
3. Выбирает технологию (SLS)
4. Выбирает сложность (Средняя)
5. Указывает количество (10 шт)
6. Видит: «от 7 500 до 12 000 ₽»
7. Опционально: сохраняет с именем клиента
```
