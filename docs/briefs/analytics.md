# Product Brief: Аналитика и дашборды

## Обзор

| | |
|---|---|
| **Название** | Аналитика ценников, расходников, ФОТ |
| **Тип** | Web-приложение (Admin panel) |
| **Приоритет** | P3 (после v1.0) |
| **Срок** | Январь 2025 |

---

## Проблема

Без данных невозможно:
- Понять реальную себестоимость
- Оптимизировать ценники
- Контролировать расходы
- Планировать загрузку

**Сейчас:** Данные разрознены или отсутствуют.

---

## Решение

Дашборды на основе накопленных данных из калькуляторов:
- Статистика расчётов
- Анализ ценников по технологиям
- Учёт расходников
- ФОТ и трудозатраты

---

## Пользователи

| Роль | Что смотрит |
|------|-------------|
| Руководитель | Общая картина, тренды, прибыльность |
| Финансист | Расходы, себестоимость, маржа |
| Технолог | Загрузка оборудования, расход материалов |

---

## Функциональные требования

### Дашборд: Обзор
- [ ] Расчётов за период (день/неделя/месяц)
- [ ] Сумма расчётов (₽)
- [ ] Конверсия: расчёт → заказ
- [ ] Топ клиентов

### Дашборд: Ценники
- [ ] Средняя цена за см³ по технологиям
- [ ] Распределение заказов по технологиям
- [ ] Динамика цен во времени
- [ ] Сравнение с рынком (ручной ввод)

### Дашборд: Расходники
- [ ] Расход материалов (кг/месяц)
- [ ] Стоимость материалов
- [ ] Прогноз закупок
- [ ] Остатки на складе

### Дашборд: ФОТ
- [ ] Часы по сотрудникам
- [ ] Стоимость часа
- [ ] Загрузка vs доступность
- [ ] Эффективность (выработка)

### Отчёты
- [ ] Экспорт в Excel
- [ ] Автоматические отчёты (еженедельно)
- [ ] Настраиваемые периоды

## Источники данных

| Данные | Источник |
|--------|----------|
| Расчёты | Таблица `quotes` |
| Заказы | Таблица `orders` (будущее) |
| Материалы | Таблица `materials` + ручной ввод |
| Расходники | Ручной ввод / интеграция |
| ФОТ | Ручной ввод |

---

## Технические решения

### Вариант 1: Кастомные дашборды
- Полный контроль
- Больше работы
- React + recharts/visx

### Вариант 2: Metabase (рекомендуется для старта)
- Быстрый старт
- SQL-запросы
- Готовые визуализации
- Self-hosted (бесплатно)

### Вариант 3: Гибрид
- Metabase для сложной аналитики
- Простой дашборд в приложении
