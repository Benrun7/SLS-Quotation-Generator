# Product Brief: Экспресс-калькулятор моделирования

## Обзор

| | |
|---|---|
| **Название** | Экспресс-калькулятор стоимости 3D-моделирования |
| **Тип** | Web + Telegram Mini App |
| **Приоритет** | P2 (средний) |
| **Срок** | 04.01.2025 |

---

## Проблема

Клиент спрашивает: «Сколько стоит смоделировать деталь по чертежу/эскизу?»

**Сейчас:** Оценка «на глаз» или «надо посмотреть» → неточно, долго.

**Нужно:** За 1 минуту дать вилку стоимости моделирования.

---

## Решение

Калькулятор оценки трудозатрат на 3D-моделирование по экспертным критериям:
- Тип изделия
- Сложность геометрии
- Наличие исходных данных
- Требования к точности

---

## Пользователи

| Роль | Сценарий |
|------|----------|
| Менеджер | Оценка для клиента без привлечения инженера |
| Инженер | Быстрая прикидка перед детальной оценкой |

---

## Функциональные требования

### Ввод данных

#### Тип изделия
- [ ] Корпусная деталь (коробка, крышка)
- [ ] Механическая деталь (шестерня, вал)
- [ ] Органическая форма (эргономика, дизайн)
- [ ] Сборка (несколько деталей)

#### Сложность
- [ ] Простая (примитивы, вытягивания)
- [ ] Средняя (скругления, отверстия, рёбра)
- [ ] Сложная (поверхности, лофты, сплайны)
- [ ] Очень сложная (органика, скульптинг)

#### Исходные данные
- [ ] 3D-скан / облако точек
- [ ] Чертёж с размерами
- [ ] Эскиз / фото
- [ ] Только описание

#### Требования
- [ ] Эскизное моделирование
- [ ] Конструкторская модель
- [ ] Производственная модель (с допусками)

### Расчёт

```javascript
// Базовые часы по типу изделия
const базаЧасов = {
  'корпусная': [2, 4],
  'механическая': [3, 6],
  'органика': [6, 12],
  'сборка': [8, 16]
};

// Множители
const множительСложности = {
  'простая': 0.7,
  'средняя': 1.0,
  'сложная': 1.5,
  'очень сложная': 2.5
};

const множительИсходных = {
  '3d-скан': 0.6,
  'чертёж': 0.8,
  'эскиз': 1.0,
  'описание': 1.3
};

const множительТребований = {
  'эскизное': 0.7,
  'конструкторское': 1.0,
  'производственное': 1.4
};

// Итого часов
const часыМин = база[0] * сложность * исходные * требования;
const часыМакс = база[1] * сложность * исходные * требования;

// Стоимость
const ставка = 1500; // ₽/час
const ценаМин = часыМин * ставка;
const ценаМакс = часыМакс * ставка;
```

### Вывод
- [ ] Диапазон часов: «от 4 до 8 часов»
- [ ] Диапазон цены: «от 6 000 до 12 000 ₽»
- [ ] Примерные сроки: «1-2 рабочих дня»

---

## Пользовательский путь

```
1. Открывает калькулятор
2. Выбирает тип изделия (Механическая деталь)
3. Выбирает сложность (Средняя)
4. Указывает исходные данные (Чертёж)
5. Выбирает требования (Конструкторское)
6. Видит: «4-6 часов, 6 000 — 9 000 ₽, 1-2 дня»
7. Сохраняет или отправляет клиенту
```
